# 🚀 PM Tool — Project Management Made Simple

A modern, fast, and simple project management tool designed for small teams to ship faster. Built with the latest Next.js 15 App Router architecture.

> 🚧 **Status:** Work in Progress  
> This project is currently under active development.

---

## 🛠 Tech Stack

- **Framework:** Next.js 15 (App Router)  
- **Language:** TypeScript  
- **Styling:** Tailwind CSS  

**Upcoming:**
- NextAuth.js v5  
- PostgreSQL  
- Prisma ORM  

---

## 📋 Features

### ✅ Currently Implemented
- Responsive Landing Page with modular architecture  
- Custom UI Components (Hero, Features, Navbar, Footer)  
- Authentication UI skeletons (Login & Registration)  

### ⏳ Planned Features
- Full Authentication (OAuth + Email/Password)  
- Protected Dashboard  
- Interactive Kanban Board  
- Team & Project Management  

---

## 🚀 Getting Started

Follow these steps to run the project locally:

```bash
git clone https://github.com/niloyhakimai/pm-tool.git
cd pm-tool
npm install
npm run dev
```

Then open your browser and go to:  
👉 http://localhost:3000
