# 🚀 PM Tool — Project Management Made Simple

A modern, fast, and simple project management tool designed for small teams to ship faster. Built with the latest Next.js 15 App Router architecture.

> **🚧 Status: Work in Progress**
> This project is currently under active development.

## 🛠 Tech Stack

* **Framework:** Next.js 15 (App Router)
* **Language:** TypeScript
* **Styling:** Tailwind CSS

*(Upcoming: NextAuth.js v5, PostgreSQL, Prisma ORM)*

## 📋 Features

### ✅ Currently Implemented
* Responsive Landing Page with Modular Architecture
* Custom UI Components (Hero, Features, Navbar, Footer)
* Auth UI Skeletons (Login & Registration)

### ⏳ Planned Features
* Authentication (OAuth & Email)
* Protected Dashboard
* Interactive Kanban Board
* Team & Project Management

## 🚀 Getting Started

To run this project locally:

```bash
git clone [https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git](https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git)
cd YOUR_REPO_NAME
